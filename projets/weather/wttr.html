Application Météo

Une application web permettant de donner la météo d'une ville donnée !

Connaitre tous les détails concernant la météo d'une ville? nous sommes là !

Dans le cadre d'un projet de groupe, nous avons décidé de créer une application web permettant de donner la météo d'une ville donnée. Nous avons utilisé l'API OpenWeatherMap pour récupérer les données météorologiques. De cette manière, vous pourez tout savoir : la température, la vitesse du vent, la pression atmosphérique, l'humidité ainsi que la température ressentie.

Technologies utilisées

HTML5

CSS3

Flask.py
