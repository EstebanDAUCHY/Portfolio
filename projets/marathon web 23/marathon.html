Marathon web 2023

Un application web appelée Le Panthéon de lire, créer, modifier et supprimer des histoires dont vous êtes le héros !

Bienvenue sur le panthéon!

Aimez vous les histoires dont vous êtes le héros? Alors vous êtes servis ! Créez, modifiez, lisez, aimer, vibrer à travers des histoires qui peuvent vous mener à faire des choix qui changeront Le cours de votre naration. En vous inscrivant, vous pourrez créer vos histoires et les partager avec la communauté du Panthéon. Vous pourrez aussi lire les histoires des autres utilisateurs et les aimer pour les faire remonter dans le classement. Vous voulez accéder à une histoire que vous avez déjà lu? Pas de problème, votre page de profil est modifiable est consultable avec les histoires que vous avez déjà lu, celles que vous avez aimé, crées ou commentées.
